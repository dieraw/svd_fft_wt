# Анализ и сравнение методов сжатия изображений  
### SVD · FFT · Wavelet (Daubechies)


- **SVD (Singular Value Decomposition)**
- **FFT (Fast Fourier Transform)**
- **Wavelet-преобразование (Daubechies, уровни 2/4/6)**

Для трёх разных изображений (портрет, улица, природа) проводится:

1. Перевод в оттенки серого  
2. Построение спектра Фурье  
3. Усечённое SVD  
4. Сжатие через FFT с сохранением части коэффициентов  
5. Wavelet-разложение с разными уровнями глубины  
6. Вычисление:  
   - нормы Фробениуса ошибки  
   - процента сохранённой энергии  
   - визуальной разницы
